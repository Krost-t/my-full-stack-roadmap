# GitHub and Git Guide 📚

This repository contains comprehensive guides for using Git and GitHub effectively. Here are the available files:

## 1. Git & GitHub Course Guide 🎓

The `course.md` file is a detailed guide covering essential aspects of Git and GitHub, including:
- An introduction to Git and GitHub, their features, and benefits.
- Steps for installing and configuring Git.
- Basic Git commands and branch management.
- Best practices for commits and conflict management.
- Advanced Git usage, including rebasing, cherry-picking, and stashing.
- Collaborating with GitHub, including creating repositories, forking, and Pull Requests.
- Automation with GitHub Actions and security best practices.

For more details, see the file: [course.md](course.md)

## 2. Git Cheatsheet 🌟

The `cheatsheet.md` file is a quick reference for the most commonly used Git commands. It includes:
- Configuration and file management commands.
- Branching and merging operations.
- Commands for working with remote repositories.
- Managing commit history and tags.
- Stashes, cherry-picking, and commit management.
- Submodules, hooks, and diff/merge tools.
- Tips and tricks for effective Git usage.

For more details, see the file: [cheatsheet.md](cheatsheet.md)
